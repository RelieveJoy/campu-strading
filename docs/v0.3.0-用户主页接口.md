# v0.3.0 用户主页接口

## 新增接口

### GET `/api/users/{id}/profile` — 用户公开主页

```js
// response.data: UserProfileVO
{
  userId: number,
  username: string,
  avatar: string | null,
  totalItems: number,        // 在售商品数 (status=1)
  totalSold: number,         // 已售出商品数 (status=2)
  overallRating: number,     // 综合评分，无评价时为0
  ratedItemCount: number     // 有评分的商品数
}

// 用户不存在: { code: 0, msg: "用户不存在", data: null }
```

## 改动接口

### GET `/api/users/{id}/items` — 用户商品列表（已改为分页）

| 参数 | 类型 | 必填 | 默认值 | 说明 |
|------|------|------|--------|------|
| status | Integer | 否 | — | 不传=全部，1=在售，2=已售出，0=下架 |
| page | int | 否 | 1 | 页码 |
| pageSize | int | 否 | 12 | 每页条数 |

```js
// 请求示例
GET /api/users/3/items                     // 全部商品
GET /api/users/3/items?status=1             // 仅在售
GET /api/users/3/items?status=2&page=1&pageSize=8  // 已售出，分页

// 返回值从 List<ItemVO> 变为 PageResult
{
  total: number,
  records: ItemVO[]   // 每项含 averageRating 和 reviewCount
}
```

## 前端数据流

```
UserHomeView (/user/:id)
  │
  ├─ GET /api/users/{id}/profile          → 头部信息 + 统计
  ├─ GET /api/users/{id}/items?page=1&...  → 分页商品列表
  └─ GET /api/reviews/users/{id}          → 综合评分（profile 已含）
```
