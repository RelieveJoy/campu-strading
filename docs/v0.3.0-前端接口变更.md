# v0.3.0 前端接口变更

## 新增：评价接口

Base: `/api/reviews`

### POST `/api/reviews` — 提交评价

```js
// request
{
  orderId: number,   // required
  itemId: number,    // required
  rating: number,    // required, 0~5。0=跳过不评, 1~5=有效评分
  content: string    // optional, 评语
}

// response: { code: 1, msg: null, data: null }
```

### PUT `/api/reviews/{reviewId}` — 修改评价

```js
// request (同 POST)
{
  orderId: number,   // required
  itemId: number,    // required
  rating: number,    // required, 0~5
  content: string    // optional
}

// response: { code: 1, msg: null, data: null }
```

### GET `/api/reviews/item/{itemId}` — 商品评价列表

```js
// response.data: ReviewVO[]
{
  reviewId: number,
  orderId: number,
  reviewerName: string,   // 评价人姓名
  rating: number,         // 0~5
  content: string | null,
  createTime: string
}
```

### GET `/api/reviews/item/{itemId}/rating` — 商品评分摘要

```js
// response.data: ItemRatingVO
{
  itemId: number,
  averageRating: number,  // 平均值，无评分为0
  reviewCount: number     // 有效评分数，rating>=1 的
}
```

### GET `/api/reviews/users/{userId}` — 用户综合评分

```js
// response.data: UserRatingVO
{
  userId: number,
  overallRating: number,    // 该用户所有售出商品的评分均值，无评分为0
  ratedItemCount: number    // 有评分的商品数量
}
```

## 业务规则（前端需要感知的）

| 规则 | 说明 |
|------|------|
| 评价入口 | 买家在订单状态为 **已完成(status=2)** 或 **已取消(status=3)** 时可以评价 |
| 评分可选 | rating=0 即"跳过不评"，写入记录但不计入均值 |
| 一笔一次 | 同订单只能评一次（后端拒重复提交） |
| 只能买家评 | 非买家调提交/修改接口返回异常 |
| 可修改 | 评价人可调 PUT 修改评分和评语 |
| 均值计算 | 所有 `averageRating` / `overallRating` 基于 `rating>=1` 计算 |

## 建议前端 API 封装

`api/review.js`:

```js
import request from '../utils/request'

export const submitReview = (data) =>
  request({ url: '/reviews', method: 'post', data })

export const updateReview = (reviewId, data) =>
  request({ url: `/reviews/${reviewId}`, method: 'put', data })

export const getItemReviews = (itemId) =>
  request({ url: `/reviews/item/${itemId}`, method: 'get' })

export const getItemRating = (itemId) =>
  request({ url: `/reviews/item/${itemId}/rating`, method: 'get' })

export const getUserRating = (userId) =>
  request({ url: `/reviews/users/${userId}`, method: 'get' })
```
